# OpenCode Agent Harness

## Mission
An OpenCode-native agent harness. It brings together the patterns that make
coding agents effective — skill-driven workflows, subagent orchestration,
standing rules, tech-aware conventions, and verification-first development.
One harness, one target, no multi-harness fragmentation.

## Tech conventions

Each name in `rules/tech.md` maps directly to `tech/<name>/` (e.g. `rust` →
`tech/rust/`). Skills use this declaration to load language-specific
conventions.

Before writing or editing code, confirm the relevant tech is in recent
context. If unsure, re-read the appropriate `tech/<name>/` files.
